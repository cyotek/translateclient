Thank you for installing Translate.Net version 0.1.3249.6795 (released at 2008-12-07).

This is free open source project with goal to simplify access to many language resources available online.

Please note, this version contains parsing errors as result of changes in servers, so please check beta version from here

Changes :

  • Added support of Microsoft's LiveTranslator - multi-language translator - 25 translation directions
  • Added support of megaslownik.pl - Polish oriented free online dictionary - Polish to English, German, Russian, French, Spanish and Italian
  • Added Greek gui translation by Spyros
  • Added Romanian gui translation by Marius Marusteri


Additional information about Translate.Net:

  • Online documentation about use of Translate.Net
  • Translate.Net also has portable version. You can download it here.
