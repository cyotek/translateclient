Возможности

Список онлайновых переводчиков и словарей, которые поддерживает Translate.Net, можно посмотреть на странице Переводчики





Интерфейс пользователя поддерживает следующие возможности:
  • история языковых пар, которые были использованы, отсортированная по времени последнего использования
  • перехват раскладки клавиатуры - когда пользователь меняет раскладку, выбирается последняя языковая пара из истории. Когда пользователь меняет языковые пары - раскладка клавиатуры меняется соответственно.
  • системные гарячие клавиши (Ctrl+C+C и\или Ctrl+Ins+Ins) позволяют активировать Translate.Net и выполнять перевод из любых программ
